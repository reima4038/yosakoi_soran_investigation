# ログイン画面スクリーンショット

このファイルは実際のログイン画面のスクリーンショット（login-screen.png）のプレースホルダーです。

## 撮影手順

1. ブラウザーでシステムにアクセス
2. ログイン画面を表示
3. 以下の要素が含まれるようにスクリーンショットを撮影：
   - ログインフォーム
   - メールアドレス入力フィールド
   - パスワード入力フィールド
   - ログインボタン
   - 新規登録リンク
   - パスワードリセットリンク

## ファイル仕様

- ファイル名: `login-screen.png`
- 推奨解像度: 1920x1080
- ファイル形式: PNG
- 最大ファイルサイズ: 2MB

## 更新頻度

UIの変更があった場合に更新してください。
