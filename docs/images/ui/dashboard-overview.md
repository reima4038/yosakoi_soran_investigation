# ダッシュボード画面スクリーンショット

このファイルは実際のダッシュボード画面のスクリーンショット（dashboard-overview.png）のプレースホルダーです。

## 撮影手順

1. システムにログイン
2. ダッシュボード画面を表示
3. 以下の要素が含まれるようにスクリーンショットを撮影：
   - ナビゲーションバー
   - サイドメニュー
   - 統計情報カード
   - 最近の活動一覧
   - クイックアクションボタン

## ファイル仕様

- ファイル名: `dashboard-overview.png`
- 推奨解像度: 1920x1080
- ファイル形式: PNG
- 最大ファイルサイズ: 2MB
