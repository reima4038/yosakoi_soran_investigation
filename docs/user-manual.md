## はじめに

YOSAKOIパフォーマンス評価システムは、YouTube上のよさこいソーラン演舞動画を複数の評価者で評価し、その結果を分析・共有するためのWebアプリケーションです。

### 主な機能
- YouTube動画の統合・管理
- カスタマイズ可能な評価基準
- 多人数による同時評価
- タイムライン連動コメント
- 詳細な分析・可視化
- 結果の共有・フィードバック

### 対応ブラウザ
- Chrome 90以上
- Firefox 88以上
- Safari 14以上
- Edge 90以上

## アカウント作成・ログイン

### 新規アカウント作成

1. **登録ページにアクセス**
   - トップページの「新規登録」ボタンをクリック

2. **必要情報の入力**
   - ユーザー名: 半角英数字、3-20文字
   - メールアドレス: 有効なメールアドレス
   - パスワード: 8文字以上、大文字・小文字・数字・特殊文字を含む

3. **メール認証**
   - 登録したメールアドレスに認証メールが送信されます
   - メール内のリンクをクリックして認証を完了

### ログイン

1. **ログインページにアクセス**
   - トップページの「ログイン」ボタンをクリック

2. **認証情報の入力**
   - メールアドレスとパスワードを入力
   - 「ログイン」ボタンをクリック

### パスワードリセット

1. **パスワードリセット申請**
   - ログインページの「パスワードを忘れた方」をクリック
   - メールアドレスを入力して送信

2. **新しいパスワードの設定**
   - メールに送信されたリンクをクリック
   - 新しいパスワードを設定

## 動画の登録・管理

### YouTube動画の登録

1. **動画登録ページにアクセス**
   - メニューから「動画管理」→「新規登録」を選択

2. **YouTube URLの入力**
   - YouTube動画のURLを入力欄に貼り付け
   - 「動画情報を取得」ボタンをクリック

3. **メタデータの入力**
   - **チーム名**: 演舞を行ったチーム名
   - **演舞名**: 演舞の名称
   - **大会名**: 開催された大会名
   - **年度**: 開催年
   - **開催地**: 大会の開催地
   - **タグ**: 検索用のキーワード（カンマ区切り）

4. **登録完了**
   - 「登録」ボタンをクリックして完了

### 動画一覧の確認

1. **動画一覧ページ**
   - メニューから「動画管理」→「一覧」を選択

2. **検索・フィルタリング**
   - **キーワード検索**: タイトル、チーム名で検索
   - **カテゴリフィルター**: 年度、大会名でフィルタ
   - **ソート**: 登録日、タイトル、アップロード日で並び替え

3. **動画の操作**
   - **詳細表示**: 動画タイトルをクリック
   - **編集**: 鉛筆アイコンをクリック
   - **削除**: ゴミ箱アイコンをクリック

### 動画情報の編集

1. **編集ページにアクセス**
   - 動画一覧から編集したい動画の鉛筆アイコンをクリック

2. **情報の更新**
   - メタデータやタグを修正
   - 「更新」ボタンをクリック

## 評価テンプレートの作成

### 新しいテンプレートの作成

1. **テンプレート作成ページ**
   - メニューから「テンプレート管理」→「新規作成」を選択

2. **基本情報の入力**
   - **テンプレート名**: わかりやすい名前を設定
   - **説明**: テンプレートの用途や特徴を記述

3. **カテゴリの追加**
   - 「カテゴリを追加」ボタンをクリック
   - **カテゴリ名**: 例）技術面、表現力、構成
   - **説明**: カテゴリの評価内容
   - **重み**: 全体評価に占める割合（0.1-1.0）

4. **評価項目の追加**
   - 各カテゴリ内で「評価項目を追加」をクリック
   - **項目名**: 具体的な評価項目名
   - **説明**: 評価の観点や基準
   - **評価方式**: 
     - 数値評価（1-10点など）
     - スケール評価（5段階など）
     - はい/いいえ評価
   - **重み**: カテゴリ内での重要度

### テンプレートの管理

1. **テンプレート一覧**
   - 作成したテンプレートの一覧表示
   - 使用回数や最終更新日を確認

2. **テンプレートの複製**
   - 既存テンプレートをベースに新しいテンプレートを作成
   - 「複製」ボタンをクリック

3. **テンプレートの共有**
   - 他のユーザーとテンプレートを共有
   - エクスポート/インポート機能を利用

## 評価セッションの実施

### 評価セッションの作成

1. **セッション作成ページ**
   - メニューから「評価セッション」→「新規作成」を選択

2. **基本設定**
   - **セッション名**: わかりやすい名前を設定
   - **説明**: セッションの目的や注意事項
   - **評価対象動画**: 登録済み動画から選択
   - **評価テンプレート**: 使用するテンプレートを選択

3. **期間設定**
   - **開始日時**: 評価開始日時
   - **終了日時**: 評価締切日時
   - **タイムゾーン**: 日本標準時（JST）

4. **詳細設定**
   - **匿名評価**: 評価者名を非表示にするか
   - **結果表示**: 提出後に他の評価を表示するか
   - **コメント許可**: タイムラインコメントを許可するか

### 評価者の招待

1. **招待ページ**
   - セッション詳細から「評価者を招待」をクリック

2. **招待方法の選択**
   - **メール招待**: メールアドレスを入力して招待
   - **リンク共有**: 招待リンクを生成して共有

3. **招待メッセージ**
   - 評価者向けのメッセージを作成
   - 評価の観点や注意事項を記載

4. **権限設定**
   - **評価者**: 評価のみ実行可能
   - **オブザーバー**: 結果閲覧のみ可能

### セッションの管理

1. **進捗確認**
   - 招待した評価者の参加状況
   - 評価完了状況の確認

2. **リマインダー送信**
   - 未完了の評価者にリマインダーメール送信

3. **期限延長**
   - 必要に応じて評価期限を延長

## 評価の実行

### 評価画面の操作

1. **評価ページにアクセス**
   - 招待メールのリンクまたはセッション一覧からアクセス

2. **動画の再生**
   - YouTube Playerで動画を再生
   - 再生速度の調整（0.5x, 0.75x, 1x, 1.25x, 1.5x, 2x）
   - 全画面表示対応

3. **評価の入力**
   - 各評価項目に点数を入力
   - 必要に応じてコメントを追加
   - 自動保存機能により途中保存が可能

### タイムライン連動コメント

1. **コメントの追加**
   - 動画再生中に「コメント追加」ボタンをクリック
   - 現在の再生時点にコメントが紐付けられる

2. **コメントの表示**
   - タイムライン上にマーカーで表示
   - マーカーをクリックすると該当時点にジャンプ

3. **コメントの編集**
   - 追加したコメントの編集・削除が可能
   - 他の評価者のコメントは閲覧のみ

### 評価の提出

1. **完了チェック**
   - すべての必須項目が入力されているか確認
   - 未入力項目がある場合は警告表示

2. **最終確認**
   - 評価内容の最終確認画面
   - 提出後は修正不可の旨を確認

3. **提出完了**
   - 「評価を提出」ボタンをクリック
   - 提出完了メールが送信される

## 結果の確認・分析

### 評価結果の概要

1. **結果ダッシュボード**
   - セッション完了後に結果ダッシュボードが表示
   - 全体的な評価傾向を確認

2. **基本統計**
   - **平均点**: 全評価者の平均スコア
   - **最高点・最低点**: 各項目の最高・最低評価
   - **標準偏差**: 評価のばらつき度合い
   - **完了率**: 評価完了者の割合

### 詳細分析

1. **カテゴリ別分析**
   - レーダーチャートでカテゴリ別スコアを表示
   - 強み・弱みの可視化

2. **評価者比較**
   - 評価者ごとのスコア比較
   - 評価傾向の違いを分析

3. **項目別分析**
   - 各評価項目の詳細分析
   - スコア分布のヒストグラム表示

4. **一致度分析**
   - 評価者間の評価一致度
   - 意見が分かれた項目の特定

### タイムライン分析

1. **コメント分析**
   - 時系列でのコメント分布
   - 注目された場面の特定

2. **評価ポイント**
   - 高評価・低評価を受けた時点の分析
   - 改善点の具体的な特定

## トラブルシューティング

### よくある問題と解決方法

#### ログインできない
**症状**: メールアドレスとパスワードを入力してもログインできない

**解決方法**:
1. パスワードの大文字・小文字を確認
2. ブラウザのキャッシュをクリア
3. パスワードリセット機能を利用
4. アカウントがロックされていないか確認

#### 動画が再生されない
**症状**: YouTube動画が表示されない、または再生できない

**解決方法**:
1. インターネット接続を確認
2. ブラウザを最新版に更新
3. 広告ブロッカーを無効化
4. 動画が削除・非公開になっていないか確認

#### 評価が保存されない
**症状**: 入力した評価が保存されない

**解決方法**:
1. インターネット接続を確認
2. ブラウザを再読み込み
3. 別のブラウザで試行
4. JavaScriptが有効になっているか確認

#### 招待メールが届かない
**症状**: 評価セッションの招待メールが届かない

**解決方法**:
1. 迷惑メールフォルダを確認
2. メールアドレスの入力ミスを確認
3. ドメイン（@yosakoi-eval.com）を受信許可設定
4. 管理者に直接連絡

### エラーメッセージ一覧

#### 認証エラー
- **「認証に失敗しました」**: ログイン情報を確認してください
- **「セッションが期限切れです」**: 再度ログインしてください
- **「アクセス権限がありません」**: 管理者に権限を確認してください

#### 入力エラー
- **「必須項目が入力されていません」**: 赤色で表示された項目を入力してください
- **「無効な形式です」**: 入力形式を確認してください
- **「文字数制限を超えています」**: 文字数を減らしてください

#### システムエラー
- **「サーバーエラーが発生しました」**: しばらく時間をおいて再試行してください
- **「ネットワークエラー」**: インターネット接続を確認してください
